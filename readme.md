## 2. Markdown 如何绘制表格 
* 格式如下：
```
| 项目|参数|数量|单价|
|:---|:---|:---|:---|
|    |    |    |    |
|    |    |    |    |
|    |    |    |    |
|    |    |    |    |
```
* 效果如下：  
| 项目|参数|数量|单价|
|:---|:---|:---|:---|
|    |    |    |    |
|    |    |    |    |
|    |    |    |    |
|    |    |    |    |
